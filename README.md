# IESO Website

Energy demand forecasting application for Ontario.

## Setup

1. **Install dependencies:**
   ```bash
   cd frontend
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

   The application will be available at `http://localhost:3000` (automatically opens in browser)

## Available Scripts

- `npm run dev` - Start Vite development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally
- `npm run check` - Type check the codebase

## Project Structure

- `client/` - Frontend React application (source code)
- `attached_assets/` - Static assets
- `dist/` - Production build output (generated)

## Notes

- Pure frontend application using Vite + React
- Vite's built-in error overlay provides runtime error reporting in development mode
- Hot Module Replacement (HMR) enabled for fast development
